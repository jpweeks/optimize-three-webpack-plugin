# @vxna/optimize-three-webpack-plugin

A compat layer that enables tree shaking without tears

## Usage

**webpack.config.js**

```js
const OptimizeThreePlugin = require('@vxna/optimize-three-webpack-plugin')

module.exports = {
  plugins: [new OptimizeThreePlugin()]
}
```
